# Features List - GitHub Coffee API

## REST API Features
- [x] **Unified Platform Endpoints**: Dedicated routes for ESM, PMP, and Foresight.
- [x] **Predictable Responses**: Standardized JSON structure across all mock data.
- [x] **Standard Endpoints**:
    - `GET /api/esm`: Energy Storage Management data.
    - `GET /api/pmp`: Piscada Management Platform data.
    - `GET /api/foresight`: Facilities Management data.

## GraphQL Features
- [x] **Dynamic Queries**: Granular data fetching for UI performance.
- [x] **Integrated Explorer**: Built-in playground for schema exploration (`/api/graphql`).
- [x] **Unified Schema**: Single schema for the entire ecosystem.

## Infrastructure Features
- [x] **TypeScript Integration**: Shared types between API and consumer platforms.
- [x] **Dynamic Mocking**: Logic-based data generation to simulate real-world scenarios.
